### 👨‍🔬 About Me

I am an **Applied Physics student** with a dual interest in **Medical Physics** and **Complexity Science**. I love using computational tools to understand how simple interactions generate the complex behaviors we see in nature, from biological tissues to algorithmic patterns.

- 🎓 I’m currently studying **Applied Physics**, focusing on simulations and data analysis.
- 🏥 **Medical Physics:** Interested in imaging, dosimetry, and biophysical modeling.
- 🕸️ **Complex Systems:** Exploring Chaos Theory and Emergence.
- 🐍 **My Tools:** Python (NumPy, SciPy, Matplotlib) for crunching numbers and visualizing data.
